## 🚀 Getting Started

### Step 1: Read Documentation
**File**: `.ai/START_HERE.md`
- 5-minute quick overview
- Next steps clearly marked

### Step 2: Download & Setup Frontend
**File**: `.ai/QUICK_START.md` → Part 1
```bash
# Download from v0.app
# Extract to your workspace
cd sabat-frontend
pnpm install
pnpm dev
# http://localhost:3000 ✓
```

### Step 3: Configure VSCode
**File**: `.ai/VSCODE_SETUP.md`
- Install extensions
- Configure settings.json
- Set up Copilot
- Configure debugging

### Step 4: Create Laravel Backend
**File**: `.ai/QUICK_START.md` → Part 2
```bash
composer create-project laravel/laravel sabat-backend
cd sabat-backend
# Follow BACKEND_SETUP.md for detailed steps
php artisan serve
# http://localhost:8000 ✓
```

### Step 5: Build with Copilot
**File**: `.ai/VSCODE_COPILOT_GUIDE.md`
- Use documented prompts
- Reference API_SPECIFICATION.md
- Follow architecture patterns
- Test endpoints with REST Client
